
## common
ニューラルネットワークの基礎的な学習プログラム、計算プログラムなどが入っている

## commonseq2seq
commonの内容を改良してSeq2seqに対応させたレイヤーが入っている

## eval
ニューラルネットワークによって作成された文書要約の評価をするためのプログラム

## layer
ニューラルネットワークの基礎レイヤーが入っている

## seq2seq_ver1
共起情報のみによる要約プログラムが入っている

## seq2seq_ver2_tf_idf
TF-IDFを考慮した要約プログラムが入っている

## seq2seq_ver3_em_tf_idf
TF-IDFと前後の発話の感情変化を考慮したプログラムが入っている
